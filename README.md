# pico-hid-mvs

Compiler: arm-gcc-none-eabi

## Setup
```
export TINYUSB_PATH=...
export PICO_SDK_PATH=...
```

## Build
```
mkdir build
cd build
cmake -DFAMILY=rp2040 ..
```
